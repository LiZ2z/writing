# Identifier

```javascript
var a = 1;
```

人 -> 变量

名字（张三，指代张三这个人） -> Identifier（**a**，指代a这个变量）

所以Identifier相对于变量，就类似名字相对于人。

我们常说“这个人的名字是张三”，也会习惯性的说“这个变量的名字是a”，但正式的说法应该是“这个变量的标志符（Identifer）是a”
