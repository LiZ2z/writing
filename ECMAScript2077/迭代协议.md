# 迭代协议

迭代协议具体分为两个协议：**可迭代协议**及**迭代器协议**。

如果一个对象实现了可迭代协议定义的接口，则被认为是**可迭代的（形容词）**，可以通过这个对象的迭代器对其进行**迭代（动词）**。

### 可迭代协议

可迭代的对象必须拥有一个@@iterator属性。

| 属性       | 值                                                         | 说明 |
| ---------- | ---------------------------------------------------------- | ---- |
| @@iterator | 一个无参数的函数，其返回值为一个符合[迭代器协议]()的对象。 |      |
