# Agent

直译：代理，代理人

user agent 就是代理用户完成一系列行为的人或工具。

在计算机中，agent既指代理用户完成一系列行为的软件或应用程序。

当我们用浏览器访问页面时，浏览器就是我们访问服务器的agent，我们不需要直接跟服务器接触，只需要输入服务器地址，浏览器会自动帮我们连接，然后接收数据、呈现。
